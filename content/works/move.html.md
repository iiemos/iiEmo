# 为什么要学习Move

## 🍊 Move语言有什么优势
> * 面向资源编程
> * 原生态避免双花攻击
> * 安全和形式化验证
> * 学习门槛较低
> * 合约可升级（相对solidity来说）

------

## Move的结构
  - Modules
    - 结构化定义
    - 函数功能
    - 全局存储
  - Scripts
    - 暂时的代码片段
    - 可以调用Modules的函数
    - 一个script中只能有一个函数
